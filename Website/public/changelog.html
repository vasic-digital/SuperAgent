Changelog

All notable changes to HelixAgent are documented here.

v1.1.0 February 2026

Major quality improvements with comprehensive test coverage expansion, performance optimizations, and security enhancements.

Fixed
  • Resolved all errcheck lint violations across internal packages (~96 fixes)
  • Fixed O(n) cache eviction in query optimizer with O(1) LRU replacement
  • Fixed race conditions detected by go test -race across codebase
  • Enabled memory service by default (decoupled from Cognee, uses Mem0)
  • Updated misleading "placeholder" comment in skills protocol adapter
Added
  • Test coverage for analytics, conversation, learning packages (previously 0%)
  • BigData package test coverage increased from 2.9% to 80%+
  • Formatters package test coverage increased from 15.5% to 80%+
  • Memory package tests: CRDT, distributed manager, event sourcing, store
  • Observability package tests: exporter, LLM middleware, metrics, tracer
  • Database query optimizer tests and O(1) LRU cache benchmarks
  • Stress tests for BigData, formatters, and memory subsystems
  • gRPC integration tests (21 tests covering all service methods)
  • Debate full-flow integration test (7 lifecycle tests)
  • Comprehensive monitoring integration tests
  • Benchmark tests for formatters cache and memory CRDT operations
  • 4 new challenge scripts: gRPC service, BigData, memory system, security scanning
  • Snyk policy configuration (.snyk)
  • SBOM generation script (CycloneDX/SPDX via syft or cyclonedx-gomod)
  • Per-package README documentation for analytics, bigdata, conversation, formatters, learning
  • 4 new user manuals: BigData integration, gRPC API, memory system, code formatters
  • 2 new video courses: BigData analytics, memory management
  • Architecture diagrams for gRPC service and memory system (Mermaid)
Improved
  • Semaphore-based concurrency limits for parallel health checks and component initialization
  • Timeout-based shutdown for all Stop/Shutdown/Close methods
  • Lazy loading for optional services (BigData, service formatters)
  • CLAUDE.md and AGENTS.md updated with new features and test references
Security
  • Snyk policy file for dependency vulnerability scanning
  • SBOM generation for supply chain security (CycloneDX + SPDX formats)
  • Gosec HIGH severity findings resolved
  • Security scanning challenge script for automated validation

v1.0.1 January 2026

Bug fixes and documentation updates for improved stability and developer experience.

Fixed
  • RAGS Challenge timeout increased from 30s to 60s for complex retrieval operations
  • CogneeService ListDatasets JSON parsing to handle both array and object responses
  • ProviderHealthMonitor mutex deadlock in updateStatus/sendAlert
  • All 147 RAGS tests now passing (100%)
Added
  • MCPS Challenge Section 9 for MCP Tool Search validation
  • SKILLS Challenge with strict real-result validation
  • Comprehensive documentation updates across all manuals
  • New video course labs (Labs 6-8)
Improved
  • Documentation coverage for challenge system
  • API reference with MCP Tool Search endpoints
  • Provider configuration guide with timeout settings
  • Video courses with 3 new hands-on labs

v1.0.0 January 2026

Initial stable release of HelixAgent with comprehensive AI orchestration capabilities.

Added
  • Multi-provider LLM orchestration with 10+ providers (Claude, DeepSeek, Gemini, Mistral, OpenRouter, Qwen, ZAI, Zen, Cerebras, Ollama)
  • AI Debate System with multi-pass validation for consensus-driven responses
  • Dynamic provider selection based on LLMsVerifier scoring
  • Semantic intent detection with zero hardcoding
  • Protocol support: MCP, LSP, ACP, OpenAI-compatible API
  • 45+ MCP server adapters (databases, design tools, AI services)
  • Comprehensive test framework with 6 test types
  • Toolkit library for building AI applications
  • Background task system with progress tracking
  • Real-time notifications (SSE, WebSocket, Webhooks)
Improved
  • Enhanced streaming support with chunked responses
  • Optimized connection pooling for MCP servers
  • Improved error handling and fallback mechanisms
Security
  • Pattern-based security scanner for vulnerability detection
  • Secure credential management for OAuth providers
  • Rate limiting and request validation middleware

v0.9.0 December 2025

Beta release with core functionality.

Added
  • Core LLM provider framework
  • Basic AI debate system
  • OpenAI-compatible API endpoints
  • Docker and Kubernetes deployment support
Fixed
  • Connection timeout issues with slow providers
  • Memory leak in streaming responses
  • Race condition in concurrent requests

v0.8.0 November 2025

Alpha release for early testing.

Added
  • Initial project structure
  • Provider interface design
  • Basic API implementation
  • Configuration system
