Changelog

All notable changes to HelixAgent are documented here.

v1.0.0 January 2026

Initial stable release of HelixAgent with comprehensive AI orchestration capabilities.

Added
  • Multi-provider LLM orchestration with 10+ providers (Claude, DeepSeek, Gemini, Mistral, OpenRouter, Qwen, ZAI, Zen, Cerebras, Ollama)
  • AI Debate System with multi-pass validation for consensus-driven responses
  • Dynamic provider selection based on LLMsVerifier scoring
  • Semantic intent detection with zero hardcoding
  • Protocol support: MCP, LSP, ACP, OpenAI-compatible API
  • 45+ MCP server adapters (databases, design tools, AI services)
  • Comprehensive test framework with 6 test types
  • Toolkit library for building AI applications
  • Background task system with progress tracking
  • Real-time notifications (SSE, WebSocket, Webhooks)
Improved
  • Enhanced streaming support with chunked responses
  • Optimized connection pooling for MCP servers
  • Improved error handling and fallback mechanisms
Security
  • Pattern-based security scanner for vulnerability detection
  • Secure credential management for OAuth providers
  • Rate limiting and request validation middleware

v0.9.0 December 2025

Beta release with core functionality.

Added
  • Core LLM provider framework
  • Basic AI debate system
  • OpenAI-compatible API endpoints
  • Docker and Kubernetes deployment support
Fixed
  • Connection timeout issues with slow providers
  • Memory leak in streaming responses
  • Race condition in concurrent requests

v0.8.0 November 2025

Alpha release for early testing.

Added
  • Initial project structure
  • Provider interface design
  • Basic API implementation
  • Configuration system
