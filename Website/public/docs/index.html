Quick Start

Get up and running with HelixAgent in minutes

Installation

Install HelixAgent using Go or Docker in just a few commands.

git clone https://github.com/helixagent/helixagent.git
cd helixagent
make build
./helixagent --config configs/development.yaml

Configuration

Configure your LLM providers and ensemble settings.

# Set environment variables
export CLAUDE_API_KEY=your_key
export DEEPSEEK_API_KEY=your_key
export GEMINI_API_KEY=your_key

Code Examples

Get started quickly with these code examples

Basic Chat Completion

curl -X POST http://localhost:8080/v1/chat/completions \
  -H "Authorization: Bearer $TOKEN" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "helixagent-ensemble",
    "messages": [
      {"role": "user", "content": "Hello!"}
    ]
  }'

Python SDK

from helixagent import HelixAgentClient

client = HelixAgentClient(api_key="your-api-key")

response = client.chat.completions.create(
    model="helixagent-ensemble",
    messages=[
        {"role": "user", "content": "Hello!"}
    ]
)

print(response.choices[0].message.content)

JavaScript SDK

import { HelixAgentClient } from '@helixagent/client';

const client = new HelixAgentClient({
    apiKey: 'your-api-key'
});

const response = await client.chat.completions.create({
    model: 'helixagent-ensemble',
    messages: [
        { role: 'user', content: 'Hello!' }
    ]
});

console.log(response.choices[0].message.content);
