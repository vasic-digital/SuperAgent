Quick Start Guide

Get HelixAgent running in under 5 minutes.

Installation

Option 1: Binary Download

# Download latest release
curl -L https://github.com/helixagent/helixagent/releases/latest/download/helixagent-linux-amd64 -o helixagent
chmod +x helixagent
./helixagent --version

Option 2: Docker

# Pull and run with Docker
docker pull helixagent/helixagent:latest
docker run -d -p 8080:8080 --name helixagent helixagent/helixagent:latest

Option 3: Build from Source

# Clone and build
git clone https://github.com/helixagent/helixagent.git
cd helixagent
make build
./bin/helixagent --version

Configuration

Create a .env file with your API keys:

# Server Configuration
PORT=8080
GIN_MODE=release

# LLM Provider API Keys (add at least one)
CLAUDE_API_KEY=your-claude-api-key
DEEPSEEK_API_KEY=your-deepseek-api-key
GEMINI_API_KEY=your-gemini-api-key

# Optional: Database
DB_HOST=localhost
DB_PORT=5432
DB_NAME=helixagent

# Optional: Redis Cache
REDIS_HOST=localhost
REDIS_PORT=6379

Your First Query

Start the server and make a request:

# Start the server
./helixagent serve

# In another terminal, make a request
curl -X POST http://localhost:8080/v1/chat/completions \
  -H "Content-Type: application/json" \
  -d '{
    "model": "helixagent",
    "messages": [
      {"role": "user", "content": "Explain quantum computing in simple terms"}
    ]
  }'

HelixAgent will route your query through the AI Debate system, combining responses from multiple providers for the best answer.

AI Debate System

The AI Debate is HelixAgent's core feature. Multiple AI providers debate to reach consensus:

# Trigger an AI debate
curl -X POST http://localhost:8080/v1/debates \
  -H "Content-Type: application/json" \
  -d '{
    "topic": "Best practices for error handling in Go",
    "rounds": 3,
    "enable_multi_pass_validation": true
  }'

How It Works

  1. Initial Response: Each AI provides their perspective
  2. Validation: Cross-validation for accuracy
  3. Polish: Refinement based on feedback
  4. Consensus: Final synthesized answer
