## Prerequisites

- Basic understanding of APIs and REST
- Familiarity with Docker (for deployment courses)
- Go programming knowledge (for development courses)
- Linux/Unix command line experience

## System Requirements

- Modern web browser
- Stable internet connection
- Docker Desktop (for hands-on exercises)
- Text editor or IDE
- 8GB RAM minimum (16GB recommended)
- Go 1.24+ (for development courses)
