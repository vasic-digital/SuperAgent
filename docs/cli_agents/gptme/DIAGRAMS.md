## Security Model

### Security Layers

```
┌─────────────────────────────────────────────────────────────────────────────┐
│                         SECURITY ARCHITECTURE                                │
├─────────────────────────────────────────────────────────────────────────────┤
│                                                                              │
│   Layer 5: User Control                                                      │
│   ┌─────────────────────────────────────────────────────────────────────┐   │
│   │  ┌─────────┐  ┌─────────┐  ┌─────────┐  ┌─────────┐                │   │
│   │  │ Approve │  │  Deny   │  │  Undo   │  │  /exit  │                │   │
│   │  └─────────┘  └─────────┘  └─────────┘  └─────────┘                │   │
│   └─────────────────────────────────────────────────────────────────────┘   │
│                                                                              │
│   Layer 4: Interactive Mode                                                  │
│   ┌─────────────────────────────────────────────────────────────────────┐   │
│   │  -y (auto-confirm)  ──►  Skip confirmations for safe tools          │   │
│   │  -n (non-interactive) ──► No user interaction possible              │   │
│   └─────────────────────────────────────────────────────────────────────┘   │
│                                                                              │
│   Layer 3: Hook System                                                       │
│   ┌─────────────────────────────────────────────────────────────────────┐   │
│   │  Pre-tool hooks ──► Pattern matching ──► Block/Allow/Modify         │   │
│   └─────────────────────────────────────────────────────────────────────┘   │
│                                                                              │
│   Layer 2: Blocklists                                                        │
│   ┌─────────────────────────────────────────────────────────────────────┐   │
│   │  Tool-specific blocklists for dangerous commands                    │   │
│   └─────────────────────────────────────────────────────────────────────┘   │
│                                                                              │
│   Layer 1: Tool Restrictions                                                 │
│   ┌─────────────────────────────────────────────────────────────────────┐   │
│   │  Read: Auto-confirm    Shell: Confirm    Save: Confirm              │   │
│   │  Browser: Confirm      Python: Confirm   Patch: Confirm             │   │
│   └─────────────────────────────────────────────────────────────────────┘   │
│                                                                              │
└─────────────────────────────────────────────────────────────────────────────┘
```
