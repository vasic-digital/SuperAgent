# TaskWeaver

## Overview

**TaskWeaver** is a Microsoft open-source coding agent for data analysis, providing an AI-powered code interpreter and plugin system.

**Repository:** https://github.com/microsoft/TaskWeaver

---

## Key Features

- Code interpreter for data analysis
- Plugin system for extensibility
- Stateful conversation
- Multi-modal support
- Secure code execution

---

## Installation

```bash
pip install taskweaver
```
