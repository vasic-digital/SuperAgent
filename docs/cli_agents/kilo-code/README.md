# Kilo-Code

## Overview

**Kilo-Code** is a TypeScript-based coding assistant forked from Cline, providing AI-powered development assistance.

**Repository:** https://github.com/Kilo-Org/kilocode

---

## Key Features

- VS Code extension
- AI-powered code completion
- Multi-provider support
- Terminal integration

---

## Installation

Available as VS Code extension.
