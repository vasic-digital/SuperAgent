### Docker Commands

```bash
# Build
docker build --rm -t gpt-engineer -f docker/Dockerfile .

# Run
docker run -it --rm \
  -e OPENAI_API_KEY="YOUR_KEY" \
  -v ./project:/project \
  gpt-engineer

# Compose
docker-compose up -d --build
```


### Pre-commit Hooks

```yaml
# .pre-commit-config.yaml

  - repo: local
    hooks:
      - id: gpt-engineer-lint
        name: GPT Engineer Lint
```
