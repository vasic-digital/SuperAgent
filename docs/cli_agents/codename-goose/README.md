# Codename Goose

## Overview

**Codename Goose** is a Rust-based AI agent framework for building extensible AI assistants.

**Repository:** https://github.com/jgenerali/codename-goose

---

## Key Features

- Rust-based implementation
- Extensible plugin system
- Multiple LLM provider support
- MCP compatibility

---

## Installation

```bash
cargo install codename-goose
```
