# Agent Deck

## Overview

**Agent Deck** is a Go-based session management CLI for Claude and Gemini, providing advanced session control and management.

**Repository:** https://github.com/asheshgoplani/agent-deck

---

## Key Features

- Multi-agent session management
- Claude and Gemini support
- Tmux integration
- Web interface

---

## Installation

```bash
curl -fsSL https://raw.githubusercontent.com/asheshgoplani/agent-deck/main/install.sh | bash
```
