# AGENTS.md - LLMOps Module

## Overview

LLMOps module (`digital.vasic.llmops`) provides tools for managing LLM evaluation pipelines, experiments, datasets, and prompt versioning.

## Key Files

- `llmops/evaluator.go` — Continuous evaluation pipeline
- `llmops/experiments.go` — A/B experiment management
- `llmops/types.go` — Core data types
